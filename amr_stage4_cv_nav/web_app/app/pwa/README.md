# PWA — клиент для вызова робота

Single-file frontend, работает в любом современном браузере и ставится на
домашний экран телефона (Add to Home Screen / Install App).

## Запуск

1. Поднять backend:
   ```bash
   cd ../backend
   pip install -r requirements.txt
   uvicorn app.main:app --host 0.0.0.0 --port 8000
   ```
2. Открыть `index.html` любым статическим сервером из `pwa/`:
   ```bash
   python3 -m http.server 8080
   ```
3. Открыть `http://<nucbox-ip>:8080/?api=http://<nucbox-ip>:8000`
   с телефона, подключённого к той же Wi-Fi сети.

Если PWA раздаётся самим FastAPI (планируется подмонтировать `pwa/` как
статику в `backend/app/main.py` на следующем шаге), параметр `?api=` не
нужен — клиент пойдёт на тот же origin.

## Что внутри

| Файл           | Назначение                                            |
|----------------|-------------------------------------------------------|
| `index.html`   | Полная страница: стили, SVG-карта, формы, WS-клиент.  |
| `manifest.json`| PWA-манифест (иконки, цвета, установка).              |
| `sw.js`        | Service worker: кэш app-shell (без API-кэша).         |

Иконки `icon-192.png` / `icon-512.png` пока не добавлены — PNG-файлы
будут сгенерированы отдельным шагом, когда определимся с фирменным
лого. До тех пор установка PWA работает, но иконка будет дефолтной.

## Цветовая палитра

Основа — сине-белая индустриальная палитра Tailwind blue 50…900 +
нейтральный slate 50…900. Акценты: зелёный `#10B981` (источник/OK),
красный `#EF4444` (ошибка), янтарный `#F59E0B` (warning).
